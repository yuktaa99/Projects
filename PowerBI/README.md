# Netflix Data Analysis with Power BI

## Overview

Interactive Power BI dashboards analyzing Netflix content trends, genres, and release patterns. Explore movies vs TV shows, popular genres, and ratings distribution.

## Dataset

* Source: [Kaggle Netflix Dataset](https://www.kaggle.com/datasets/shivamb/netflix-shows)
* Includes Title, Type, Genre, Release Year, Rating, Duration, Country

## Features

* Interactive dashboards with slicers and bookmarks
* Trend analysis of content over years
* Visual insights on genre, type, and country distribution

## Tools

Power BI Desktop, Power Query, DAX


## Insights

* Drama & Comedy dominate Netflix content
* TV shows have increased over time
* Ratings vary across regions
